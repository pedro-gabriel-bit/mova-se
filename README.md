# 🏃 Mova-se — Site de Atividades Físicas

Projeto de site voltado para incentivo à prática de atividades físicas, desenvolvido durante o curso Técnico em Informática (Senac).

## 📄 Páginas
- `index.html` — Página inicial
- `pacotes.html` — Pacotes e planos disponíveis
- `acessorio.html` — Acessórios e equipamentos
- `contato.html` — Formulário de contato

## 🛠️ Tecnologias
- HTML5

## 🎯 O que aprendi
- Estruturação de páginas com HTML puro
- Navegação entre múltiplas páginas
- Organização de conteúdo com tags semânticas
